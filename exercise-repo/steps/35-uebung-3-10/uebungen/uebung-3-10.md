# Übung 3-10: Mehrere Agenten gleichzeitig

Beim Einsatz von Agenten wartet man oft, bis Aufgaben erledigt sind. Diese Wartezeit können wir nutzen, um weitere Agenten zu bedienen. Das sollte uns einen weiteren enormen Effizienzgewinn einbringen. Damit sich die Agenten nicht in die Quere kommen, können wir das Work-Tree-Feature von Git nutzen.

## Aufgabe

- Erzeuge einen zusätzlichen Work-Tree und starte Claude Code sowohl im normalen Projekt als auch im Work-Tree.
- Entwickle mit Claude Code parallel an den restlichen User-Stories aus dem Backlog.
- Überlege dir dazu für jede Story sinnvolle reviewbare Einheiten und setze jede dieser reviewbaren Einheiten mit einem eigenen Agenten um.

## Hinweis

Damit ihr nicht so viel von Agent zu Agent springen müsst, könnt ihr die Agenten wieder im YOLO-Mode starten und nur, wenn er Hilfe benötigt, assistieren.
