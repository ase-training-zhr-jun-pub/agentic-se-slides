# Standort auswählen

## User Story
Als INNOQ-Mitarbeiter möchte ich einen Bürostandort auswählen, damit ich Räume am gewünschten Standort finden kann

## Beschreibung
Diese User Story ermöglicht es INNOQ-Mitarbeitern, einen der acht Bürostandorte (Monheim, Berlin, Hamburg, Köln, München, Zürich, Cham, Offenbach) auszuwählen. Die Standortauswahl ist der erste Schritt im Prozess der Raumbuchung und bildet die Grundlage für die Anzeige der verfügbaren Räume am gewählten Standort. Die Funktion unterstützt die Multi-Standort-Unterstützung von Calvin und ermöglicht flexible standortübergreifende Zusammenarbeit.

## Akzeptanzkriterien
- [ ] Alle acht INNOQ-Standorte (Monheim, Berlin, Hamburg, Köln, München, Zürich, Cham, Offenbach) werden zur Auswahl angezeigt
- [ ] Der Mitarbeiter kann genau einen Standort aus der Liste auswählen
- [ ] Nach der Standortauswahl wird der gewählte Standort dem Benutzer klar angezeigt
- [ ] Die Standortauswahl kann jederzeit geändert werden
- [ ] Die Benutzeroberfläche zeigt die Standorte in einer benutzerfreundlichen und übersichtlichen Form an
