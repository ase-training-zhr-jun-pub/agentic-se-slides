# Übung 3-4: Vibe Coding

Es gibt jetzt eine neue Readme im Backlog-Ordner. In dieser Readme wird der Backlog sortiert und der Scope des aktuellen Sprints definiert.

Claude Code kann man auch im sogenannten YOLO-Mode starten. Dann fragt der Agent nicht mehr nach, ob er etwas darf, sondern tut es einfach.

Kommando zum Starten im YOLO-Mode:

`claude --dangerously-skip-permissions`

## Aufgabe

1. Committe deine aktuellen Changes auf main.
2. Wechsle auf einen neuen Branch `vibe-coding`.
3. Starte Claude Code im YOLO-Mode.
4. Vibe Code den aktuellen Sprint (Backend-API inklusive Anbindung ans Frontend).

Docs:

https://docs.claude.com/en/docs/claude-code/devcontainer
