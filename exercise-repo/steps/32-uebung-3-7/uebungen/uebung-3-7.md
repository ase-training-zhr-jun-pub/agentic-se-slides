# Übung 3-7: Pull Request

Claude Code kann für uns auch "schöne" Pull-Requests erstellen. Das wollen wir jetzt ausprobieren.

## Aufgabe

1. Bleibe in der Claude-Code-Session, oder (falls du sie schon geschlossen hast) öffne die letzte Session, in der du den Subtask umgesetzt hast.
2. Lasse Claude Code mit dem `gh` CLI Tool einen Pull-Request für deine Änderungen erstellen.

Wichtig: Benutze deinen eigenen Remote zum Pull Request erstellen und nicht den der Orga.
