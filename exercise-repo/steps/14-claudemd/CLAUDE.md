Diese Datei bietet Orientierung für Claude Code (claude.ai/code) bei der Arbeit mit Code in diesem Repository.

## Projektüberblick

Calvin ist INNOQs internes Raum- und Arbeitsplatzbuchungssystem zur Verwaltung von Ressourcen an 8 Bürostandorten (Monheim, Berlin, Hamburg, Köln, München, Zürich, Baar, Offenbach). 

## Struktur
- `docs/`: Umfangreiche Dokumentation. Lies die Dokumentation, wenn du etwas nicht weißt.
- `frontend/`: Das Calvin Frontend als React App
- `backend/`: Das Calvin Backend Projekt

## Vorgehen
Erstelle beim Coden in regelmäßigen Abständen kleinteilige Commits, damit Änderungen, die in eine falsche Richtung gehen im Zweifel Rückgängig gemacht werden können.
