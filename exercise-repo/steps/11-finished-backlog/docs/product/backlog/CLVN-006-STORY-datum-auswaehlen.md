# Datum auswählen

## User Story
Als INNOQ-Mitarbeiter möchte ich ein gewünschtes Datum auswählen können, damit ich einen Raum für den benötigten Zeitpunkt buchen kann

## Beschreibung
Als INNOQ-Mitarbeiter benötige ich die Möglichkeit, im Calvin Buchungssystem ein spezifisches Datum auszuwählen, um eine Raumbuchung für einen konkreten Zeitpunkt vorzunehmen. Die Datumsauswahl ist der erste Schritt im Buchungsprozess und bildet die Grundlage für die weitere Verfügbarkeitsprüfung und Raumauswahl. Die Funktion soll intuitiv bedienbar sein und eine schnelle Navigation zu gewünschten Terminen ermöglichen.

## Akzeptanzkriterien
- [ ] Ein Kalender-Widget zur Datumsauswahl ist in der Benutzeroberfläche integriert
- [ ] Der Kalender zeigt standardmäßig das aktuelle Datum an
- [ ] Nach Auswahl eines Datums wird dieses in einem lesbaren Format angezeigt (z.B. "14.10.2025")
- [ ] Das gewählte Datum wird in der Session gespeichert und bleibt bei Navigation erhalten

- [ ] Min. 1 Unit Test geschrieben
