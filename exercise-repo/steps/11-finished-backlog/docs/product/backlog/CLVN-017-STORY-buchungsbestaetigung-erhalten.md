# Buchungsbestätigung erhalten

## User Story

Als INNOQ-Mitarbeiter möchte ich eine Buchungsbestätigung erhalten, damit ich die erfolgreiche Reservierung verifizieren kann

## Beschreibung

Nach dem erfolgreichen Absenden einer Raumbuchung benötigen Mitarbeiter eine klare Bestätigung, dass die Reservierung erfolgreich im System gespeichert wurde. Diese Bestätigung soll alle relevanten Buchungsdetails enthalten und dem Mitarbeiter Sicherheit geben, dass der Raum zum gewünschten Zeitpunkt reserviert ist. Die Buchungsbestätigung ist der finale Schritt im Buchungsprozess und schließt die User Journey ab.

## Akzeptanzkriterien

- [ ] Nach erfolgreichem Absenden der Buchung wird automatisch eine Bestätigungsseite/Bestätigungsmeldung angezeigt
- [ ] Die Bestätigung enthält alle relevanten Buchungsinformationen:
  - [ ] Standort des gebuchten Raums
  - [ ] Name des gebuchten Raums
  - [ ] Datum der Buchung
  - [ ] Zeitraum der Buchung (Start- und Endzeit)
- [ ] Ein visueller Indikator (z.B. Häkchen-Symbol, grüner Hinweis) signalisiert den erfolgreichen Abschluss
- [ ] Die Bestätigung bietet eine Möglichkeit, zur Übersicht zurückzukehren oder eine weitere Buchung vorzunehmen
- [ ] Fehlerhafte Buchungen (z.B. aufgrund von Doppelbuchungen oder Systemfehlern) werden klar kommuniziert mit entsprechender Fehlermeldung
- [ ] Die Bestätigung ist responsiv und auf verschiedenen Endgeräten nutzbar

## Definition of Done

- [ ] Akzeptanzkriterien erfüllt
- [ ] Min. 1 Unit Test geschrieben
- [ ] Manuell getestet
- [ ] Dokumentation geupdated
