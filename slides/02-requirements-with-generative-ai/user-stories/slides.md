# What are User Stories?

<div class="space-y-6 text-xl leading-relaxed">
  <p><span class="font-bold">Origin:</span> Agile Software Development, popularized by Extreme Programming (Kent Beck)</p>
  <p><span class="font-bold">Purpose:</span> Help teams understand what a user actually needs by describing requirements from the user's perspective. They keep discussions focused on value instead of technical details, create a shared understanding, and support iterative development by breaking work into small, testable slices.</p>
  <p><span class="font-bold">Recommended Input:</span> User story maps, UX flows, proto-personas, technical requirements, business rules, acceptance criteria guidelines</p>
</div>
