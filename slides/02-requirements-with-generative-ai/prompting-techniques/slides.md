# Zero Shot Prompt

- Model gets task without any examples
- Relies purely on pre-trained knowledge
- Hope the result will match expectations
- Works well for common, clear tasks

## When to use

- Simple, straightforward tasks
- Standard operations (translate, summarize, classify)
- When examples would be redundant
- Exploration

# One Shot / Few Shot Prompt

- Provide the LLM with a task
- Add 1-5 examples of desired output
- Model learns pattern and applies it
- More control over format and style
- Good when you exactly know what you want

## When to use

- Custom formats or structures
- Specific requirements, like tone or style
- Edge cases to handle
