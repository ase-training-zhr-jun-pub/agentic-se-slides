# Counter-steer, on purpose

<div class="grid grid-cols-3 gap-4 mt-8 text-center">
  <div class="bg-lightGray text-petrol p-5">
    <div class="text-3xl mb-2">📝</div><b>Make intent explicit</b><div class="text-sm mt-1">the why isn't there yet, write it where the agent can reach it</div>
  </div>
  <div class="bg-lightGray text-petrol p-5">
    <div class="text-3xl mb-2">🧠</div><b>Keep the theory</b><div class="text-sm mt-1">make sure shared understanding doesn't quietly slip past or leak away</div>
  </div>
  <div class="bg-lightGray text-petrol p-5">
    <div class="text-3xl mb-2">🔁</div><b>Feed the loops back</b><div class="text-sm mt-1">harvest learnings from reviews into the system</div>
  </div>
</div>

<div class="mt-5 text-center text-base text-petrol">And you can use AI for the counter-steering too. It doesn't all have to be manual.</div>
